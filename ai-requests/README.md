# ai-requests/

Claude / Codex / Claude Code Desktop への依頼文を保管するディレクトリ。

## 命名規則

YYYY-MM-DD_<相手>_<目的>.md

- 相手: codex / claude-code / claude-ai
- 目的: 動詞ベースの簡潔な英語(review / fix / roadmap / handoff 等)

## サブディレクトリ

- ルート直下: 公開可能な依頼文
- local/: 個人情報を含むなど public 公開不可のもの(.gitignore で除外)

## 変更要求・機能要望の記録方法

大会運営中に気づいた問題や要望は、以下の5項目でメモしてください。

1. 誰が（運営者・参加者など役割で）
2. いつ（ラウンド何局目、大会開始前/後など）
3. 何に困ったか
4. 何をしたかった
5. 代替運用で乗り切れたか

記録場所: ai-requests/local/ に YYYY-MM-DD_要望メモ.md として保存
公開不要なためローカル限定（.gitignore で除外済み）
