月例将棋大会 当日マニュアル

1. 大会開始前
https://kazuo1970takahashi-sketch.github.io/shogi/
  • 参加者を登録する
  • 「JSONバックアップ保存」を押す
  • 「大会開始」を押す
2. 各ラウンド（毎回）
  • 「ペアリング生成」で組み合わせ確認
  • 対局実施 → 勝者を入力
  • 「ラウンド確定」を押す
  • 「JSONバックアップ保存」を押す
保存すると shogi_YYYYMMDD_HHMM.json がダウンロードされます。ファイルの存在を必ず確認してください。
3. 大会終了後
  • 「成績」タブで順位確認
  • 「JSONバックアップ保存」を押す（最終）
  • 印刷 または 報告書ダウンロード
4. トラブル時
ブラウザが落ちた 「読み込み」→ 直前の JSON を選択
端末が壊れた 別端末でアプリを開き同じ JSON を「読み込み」
JSON が見当たらない ダウンロードフォルダで「shogi_」で検索
ℹ️ 連絡先は別途ご確認ください。
