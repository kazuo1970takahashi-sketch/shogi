沼津支部 月例将棋大会 当日運営マニュアル

1. 大会開始前

  • アプリを開く：https://kazuo1970takahashi-sketch.github.io/shogi/
  • 参加者を登録する（名前・支部員区分・学年）
  • 「JSONバックアップ保存」を押す（ファイル名を確認してダウンロード完了を確認）
  • 「大会開始」を押す

2. 各ラウンド（毎回繰り返す）

  • 「ペアリング生成」を押して組み合わせを確認
  • 対局を実施する
  • 勝者をアプリに入力する
  • 「ラウンド確定」を押す
  • 「JSONバックアップ保存」を押す（毎ラウンド必須）
⚠️ 保存を忘れると、ブラウザを閉じたときにデータが消えます。
保存すると「shogi_YYYYMMDD_HHMM.json」というファイルがダウンロードされます。

3. 大会終了後

  • 「成績」タブで順位を確認
  • 「JSONバックアップ保存」を押す（最終バックアップ）
  • 必要に応じて「印刷」または「報告書ダウンロード」を実行

4. トラブル時の復旧手順

状況 対処
ブラウザが落ちた・誤ってリセットした 「読み込み」ボタン → 直前に保存した JSON ファイルを選択
端末が壊れた・バッテリー切れ 別の端末でアプリを開き、同じ JSON ファイルを「読み込み」で復元
JSON ファイルが見当たらない 端末のダウンロードフォルダを確認（shogi_で始まるファイル）
ℹ️ 連絡先は別途ご確認ください。
