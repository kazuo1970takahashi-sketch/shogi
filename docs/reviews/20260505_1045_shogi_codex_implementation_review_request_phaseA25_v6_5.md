## 6. 実機での動作確認

```bash

git checkout feat/phase-a25-terminology
bash test/run_tests.sh shogi_v4.html
```

### 実機テストのシナリオ（推奨）

1. **保存 → 貼り付けマイグレの一貫性**
   - shogi_v4.html を開く → サンプル大会データを作成 → 「大会データをコピー」をタップ
   - クリップボードにコピーされる → 「マスタ」タブ → 「📥 過去大会を支部マスタに統合」
   - テキストエリアに貼り付け → 「統合を実行」 → 統合成功

2. **複数大会の空行区切り**
   - 2つの大会データをそれぞれコピー → 空行で区切って連続貼り付け
   - 「2大会を読込」と表示されること

3. **エラーケース**
   - 空欄で統合 → 「大会データを貼り付けてください」表示
   - 不正テキスト貼り付け → 「形式が正しくありません」表示
