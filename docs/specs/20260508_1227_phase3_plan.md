### 1.3 既存「過去参加者から選ぶ」アコーディオンパターン

`bindPastParticipantsToggle`(L1794-1803):
```js
function bindPastParticipantsToggle(){
  var btn=document.getElementById('ppToggleBtn');
  var panel=document.getElementById('ppPanel');
  if(!btn||!panel)return;
  btn.addEventListener('click',function(){
    var open=panel.style.display!=='none';
    if(open){panel.style.display='none';btn.textContent='▼ 開く';}
    else{panel.style.display='block';btn.textContent='▲ 閉じる';}
  });
}
```

**パターン**: ボタン textContent 切替(▼/▲)+ panel display 切替(`none`/`block`)。シンプルで自前再実装可能。

#### 流用可否判断

- **関数自体の流用**: 不可(`bindPastParticipantsToggle` は init() 時に 1 度 bind されるが、F7 モーダルは開く度に DOM が再生成されるため、F7 内の bind は `bindMasterEditModalEvents` 内で local bind が必要)
- **パターン流用**: 可。同じ textContent 切替ロジック / display 切替を F7 用に再実装する(数行で済む)
