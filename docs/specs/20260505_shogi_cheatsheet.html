1
受付・参加者登録
  • 参加者に対局カードを記入してもらう
    （氏名・クラス・支部員・一般/中学）
  • 集金する
  • アプリの「参加者を登録する」タブを開く
  • 氏名を入力 → NEW 過去参加者なら候補が出るのでタップ
  • クラス（A/B）を選択して「追加」
  • 追加後の行で支部員・中学生区分を確認・変更
過去参加者はサジェストをタップするだけで氏名・クラス・区分が自動入力されます
奇数になったら運営者が参加して偶数に調整する
2
対局管理（全4局）
  • 全員登録後「登録完了・対局開始」を押す
  • 1回戦ペアリングが自動表示される
  • 対局カードに対戦相手を記入する
  • 対局終了 → 勝者のボタンをタップ
  • 「確定して次へ」で次回戦ペアリングを生成
  • 2〜4局目も同じ手順で繰り返す
ペアリングルール
2局目以降：同じ勝ち数の人同士
再戦は自動で回避されます
入力ミスは「修正」ボタンで変更できます
3
最終結果・表彰
  • 4局終了後「最終結果」タブを開く
  • A・B各クラスの順位が自動表示される
  • 順位確定基準：勝数 → SOS → SODOS
  • この画面を見ながら表彰式を行う
  • 「大会データをコピー」ボタンを押す
  • コピーしたデータは次回大会前にマスタタブで取り込む
コピー後は次回まで保管。紙への転記はアプリ画面を見ながら後で行えます
★
マスタ機能（次回大会前に実施）NEW
  • 「マスタ」タブを開く
  • 「過去大会を支部マスタに統合」をタップ
  • 前回大会のデータを貼り付けて「統合」
  • 次回から参加者がサジェストで呼び出せる
マスタには氏名・ふりがな・クラス・支部員区分・参加回数が蓄積されます
マスタタブでできること
✅ 氏名・ふりがな編集
✅ 参加者の削除（一覧から非表示）
✅ バックアップのエクスポート・インポート
✅ 前回参加・常連などで絞り込み
!
よくある操作・困ったとき
  • 結果を間違えた
    対局行の「修正」ボタンで変更できます
  • 参加者を削除したい
    登録リストの削除ボタン（対局開始前のみ）
  • クラスを変更したい
    一度削除して再登録する
  • データが消えた
    ページを閉じるとリセットされます。必ず「大会データをコピー」を押してください
  • 印刷したい
    ブラウザの印刷機能を使用（A4対応）
